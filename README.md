# Pico GitHub Activity Plugin

## Requirements 
- jQuery 
- moment.js (included via CDN)

## Install
- Copy pico_githubactivity.php to your /plugins directory
- Insert {{ GitHub }} in your template. 
- Configure 

## Settings
- `$config['GitHub']['username'] = '';`
- `$settings['GitHub']['posts'] = '';`
